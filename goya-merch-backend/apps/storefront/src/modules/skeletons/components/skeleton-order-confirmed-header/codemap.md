# goya-merch-backend/apps/storefront/src/modules/skeletons/components/skeleton-order-confirmed-header/

## Responsibility
Placeholder for the order confirmation page header (order number, date, status badges).

## Design
Renders three lines of placeholder bars with varying widths (2/5, 3/6, and two small 16/12-width bars) inside a flex column with `animate-pulse`.

## Integration
Consumed by `SkeletonOrderConfirmed` template.
