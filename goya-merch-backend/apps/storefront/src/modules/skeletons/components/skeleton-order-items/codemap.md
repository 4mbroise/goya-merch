# goya-merch-backend/apps/storefront/src/modules/skeletons/components/skeleton-order-items/

## Responsibility
Placeholder for the list of purchased items on the order confirmed page.

## Design
Renders three repeated "rows" (hardcoded), each with a 122px-wide image placeholder (29/34 aspect ratio) and text bars for title, SKU, quantity, and price in a two-column grid.

## Integration
Consumed by `SkeletonOrderConfirmed` template.
