# goya-merch-backend/apps/storefront/src/modules/skeletons/components/skeleton-card-details/

## Responsibility
Placeholder for a card details form section (e.g., payment card input fields with label).

## Design
Renders a label bar (`w-1/4`) and a full-width input field placeholder using `bg-ui-bg-field` border styling with `animate-pulse`.

## Integration
Used as loading fallback for card details forms in the checkout flow.
